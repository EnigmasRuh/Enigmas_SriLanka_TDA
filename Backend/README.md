## Setup Instructions

To set up the backend of the Book Social Network project, follow these steps:

1. Navigate to the backend directory:

```bash
  cd book-social-network
```

2. Install dependencies:

```bash
  npm install
```

4. Run the application

```bash
  npm run start
```
